Link opencart, is a wordpress plugin that connects to the opencart database and displays random products in your wordpress site. 



== Description ==

[Link opencart](http://www.webkix.com.au/) is a basic wordpress plugin that allows to link your opencart website database and display some random active products in your wordpress blog or website. This plugin can be used as a sidebar widget. With this version the wordpress and opencart should be on the same localhost or the opencart database should be accessible through the wordpress to connect and display the information. You will also need the opencart database details (you can copy from the config.php file in the opencart installation folder). 

[Visit for latest version and updates](http://www.webkix.com.au/)


Features include:



*  Show random active products from the opencart website.

*  A widget to include in the sidebar or footer to display the products.

*  Assign how many product to display from the widgets.



== Installation ==



This section describes how to install the plugin and get it working.





1. Download the plugin files

2. Extract the files in to the wordpress plugin directory (wp-content/plugins)

3. Activate the plugin from the wordpress Plugins > Installed plugins

4. Update the settings under Settings > Link opencart

5. Configure the widget under Appearance > Widgets
